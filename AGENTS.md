# Agent Guidelines — Huffman Compression Project

## Project Goal
Implement a complete **Huffman compression/decompression** tool in C with a focus on robustness, clarity, and efficient bit-level I/O.

## Current Phase
**Phase 2: Edge Cases & Robustness**

## Immediate Next Steps
1.  Handle single-symbol input more robustly.
2.  Validate header integrity on decompression.
3.  Add `--verbose` flag for progress/logging.
4.  Address compiler warnings (`-Wall -Wextra`).
